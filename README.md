
# Notes app

## Prerequisites
- MongoDB
- Node
- npm

## Server-side usage (PORT: 3000)
```terminal
$ cd server
$ npm i
$ npm start
```

## Client-side usage (PORT: 8000)
```terminal
$ cd client
$ npm i
$ npm start
```
